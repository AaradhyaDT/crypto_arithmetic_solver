# Crypto Arithmetic Solver

See `src/crypto_arithmetic_solver.py` for the solver implementation. Scripts, tests and reports were reorganized as follows:

- `src/` — implementation and FastAPI app
- `tools/` — utility scripts (puzzle checks, JSON export, report generator)
- `tests/` — lightweight test runner
- `reports/` — generated JSON and markdown reports
- `docs/` — documentation

Run `tools/check_and_export.py` to regenerate the JSON and Markdown report.
